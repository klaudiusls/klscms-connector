# KLS CMS Connector

WordPress plugin to connect a WordPress site with the KLS CMS Portal.

## Features

- Secure API key authentication
- Content sync with KLS CMS
- Schema-driven content system
- Rate limiting protection
- WordPress REST integration

## Installation

1. Download plugin
2. Upload to WordPress
3. Activate plugin
4. Go to Settings → KLS CMS
5. Paste API Key from portal

## Requirements

- WordPress 6+
- PHP 8+
